# Screenshot Naming Guide

Save your screenshots with these exact names in the `screenshots/` folder:

1. **sco-smb-hero-dashboard.png** - Main dashboard view (the one with gradient hero)
2. **sco-smb-dashboard-detail.png** - Dashboard with cards/stats
3. **sco-smb-security-settings.png** - Security settings page (enterprise features)
4. **sco-smb-scan-history.png** - Scan history with thumbnails
5. **sco-smb-ftp-server.png** - FTP server configuration
6. **sco-smb-general-settings.png** - General settings page
7. **sco-smb-splash-screen.png** - App splash screen with loading
8. **sco-smb-logo-illustration.png** - Logo with businessman illustration

After saving, delete this file.
